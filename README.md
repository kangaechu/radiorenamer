# radiorenamer

## Overview

radigoで取得した ファイル名 `<yyyymmddhhmmss>-<station>.aac`、または`<yyyymmddhhmmss>-<station>-<areaid>.aac`から

- radikoから番組情報取得
- ファイル名をリネーム
- タグを付与
- m4aとして保存

します。

## Usage

```
$ radiorenamer <yyyymmddhhmmss>-<station>.aac
or 
$ radiorenamer <yyyymmddhhmmss>-<station>-<areaid>.aac
```
